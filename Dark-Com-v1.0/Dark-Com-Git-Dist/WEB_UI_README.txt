DARK-COM BROWSER ENGINE SETUP
-----------------------------
To run the included Dark-Com.exe, you must add the WebKit engine manually:

1. Download the 'Full Release' zip from the GitHub Releases tab.
2. Copy 'jfxwebkit.dll' and all .jar files into this folder.
3. Run Dark-Com.exe.

This 'light' version only contains the UI logic and executable.
